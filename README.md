# DKNd

Python daemon for integrating a BRP069A62 Daikin Altherma LAN adapter with home assistant


*Requirements:*
 - python3 
 - python libraries:
   * requests,
   * pyyaml
   * websocket-client




*Sources:*

Using the WS endpoints was learned from  posts in these forums:
